The short answer

The strongest completed sentence KPI is STS Spearman 0.8014 from RG/UniGemma, but that run kept the full encoder body. The strongest actual compression pilot is STS 0.2995 from HG2 PCA + task contrastive training; it stopped early and is not a finished multi-task result.

02 · The diagnosis

Three lessons that generalize

The experiments failed for different operational reasons, but the recurring mechanism is simple: compression changed the function before the loss had a chance to teach it back.

01 / REPRESENTATION

Width slicing changes the basis

Taking the first 384 coordinates from a 1024-wide model is not equivalent to training a 384-wide encoder. The original basis distributes semantic information across coordinates, heads, and intermediate channels.

02 / COMPUTATION

Layer pruning changes the algorithm

Removing layers removes sequential transformations, not just parameters. Final-vector MSE can align an output while leaving the internal route that produces robust neighborhoods fundamentally broken.

03 / OBJECTIVE

Geometry is not a task KPI

Low vector error, high PCA variance, or near-perfect token-row cosine can coexist with weak STS. Retrieval, clustering, classification, and STS must each be measured and trained for explicitly.

03 · Forward path

The next credible compression route

Use the quality runs as ceilings and controls; use the native-width 270M plan as the actual compression experiment.

Harrier 270M → ~120M

Preserve the learned hidden basis at d=640, then spend the parameter budget on deliberate layer and vocabulary pruning.

  • Establish teacher and fresh-pruned baselines on the same multilingual task slice.
  • Prioritize vocabulary rows from the support-language corpus and validate tokenizer behavior.
  • Distill final embeddings first; keep layerwise loss off when teacher and student tokenization diverge.
  • Add balanced task-aware contrastive data only after the fresh student passes smoke evaluation.

Promotion gate

A checkpoint is useful only if the evidence survives outside the training loss.

  • SentenceTransformer export reports the intended embedding dimension.
  • STS improves over the fresh pruned control.
  • At least three of STS, retrieval, clustering, and classification are preserved or improved.
  • Checkpoint, corpus, and normalized-loss metadata are reproducible and resumable.
